# CyberLab Status Checker

A Python-based network reconnaissance tool that checks the availability of a website by retrieving its IP address, HTTP status code, server information, and response time.

## Features

- Resolve domain names to IP addresses
- Measure HTTP response time
- Display HTTP status code
- Show server information
- Handle connection errors gracefully

## Technologies

- Python 3
- socket
- urllib
- time
- urllib.parse
